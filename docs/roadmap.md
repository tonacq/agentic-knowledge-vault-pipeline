- [ ] Select an owned, authorised or openly licensed sample corpus
- [ ] Publish a small finished demonstration vault

- [ ] Generalise the manifest beyond YouTube-specific fields

- [ ] Add explicit, authorised webpage ingestion

- [ ] Add deployment, scheduling and recovery guidance

## Phase 4 — Usability

- [ ] Provide a guided installer
- [ ] Add pre-flight checks and actionable error messages
- [ ] Publish a public knowledge-base interface for non-technical users
